CGLocalization

If you prefer, instead of Generate a Script of Constants, you can use the SelectKeyDropdown attribute to select the identifier or the name of a key in the Inspector using a dropdown.

Let's see next example script:


using UnityEngine;
using CGLocalization;

public class Test : MonoBehaviour
{
    // select a key by id and must be a String key
    [SelectKeyDropdown(KeyType.String)]
    public int keyId;
	
    // select a key by name, can be any key type
    [SelectKeyDropdown]
    public string keyName;

    // select a key by id and must be an Audio key
    [SelectKeyDropdown(KeyType.Audio)]
    public int audioKey;

    // the attribute is only used with integers and strings
    [SelectKeyDropdown]
    public float key;
}
