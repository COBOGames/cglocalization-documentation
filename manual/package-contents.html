CGLocalization

After you have imported the package from the Asset Store into your project, the tool is installed at the Assets/CGLocalization folder.

You can move the CGLocalization folder to any location inside the Project View.

The next table show the package contents:

Folder Description
Documentation User Manual and Scripting API documentation
Editor Editor classes and resources
Helpers Various helpers classes
InspectorEditors Classes for Custom Inspector Editors
Others Other classes
PopupWindows Classes for auxiliary Popup Windows used by the main window
PropertyDrawers Classes for Custom Property Drawers
Resources Resources needed by the main window (These resources are not included when you build your project)
Windows Classes for auxiliary Editor Windows
Examples Example scenes and the assets required to play them
Resources Runtime resources used by the tool
Runtime Runtime classes
Core CGLocalization core classes
LocalizedBehaviours Localized behaviours for UGUI, TextMesh Pro and other Unity standard components
