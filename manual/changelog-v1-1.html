CGLocalization

Released 13 Jan 2020

FIXED
  • Editor: Optimized the 'Add Language' and 'Languages' windows when rendering a lot of items inside the scroll view.
  • Editor: While searching and a key name was changed, the new name was assigned to another key.
  • Editor: While searching and a key was duplicated, another key was duplicated instead of the wished key.
  • Editor: When translating texts with more than one lines (with Google API), only the first line of the translated text was used.

ADDED
  • LocaleHelper class. Helper class for ISO 639-1 language codes.
  • Localization.CURRENT_LANGUAGE_NAME_PREF_KEY
  • Localization.currentLanguageName
  • Localization.currentLanguageLocale
  • Localization.IsSupportedLocale(string)
  • LocalizationAsset.GetLanguageByLocale(string)
  • LocalizationSettings.testLanguage
  • LocalizationSettings.SINGLETON_PATH
  • LocalizationSettings.rememberCurrentLanguageName
  • The 'Locales.txt' file in the 'Resources/CGLocalization' folder.
  • Editor: 'Languages' Window: Now you can sort languages by dragging and see its options by clicking the Options button at the right.

CHANGED
  • Moved the 'Runtime' and 'Editor' folders to the 'Core' folder.
  • Moved the 'CGLocalization User Manual.pdf' and 'CGLocalization.chm' files to the 'Documentation' folder.
  • Moved the 'LocalizationSettings.asset' file to the 'Resources/CGLocalization' folder.
  • Editor: Now you can add to your LocalizationAsset more languages than those in the SystemLanguage enumeration and also custom ones.
  • Editor: The 'Select Language' window was renamed to 'Languages'.
  • Editor: 'Add Language' Window: Shown all the languages listed in the LocaleHelper.NAMES and LocaleHelper.LOCALES sorted by locale.
  • Editor: The LocalizationSettings was separated from the CGLocalization window settings.

DEPRECATED
  • Localization(LocalizationAsset, bool) constructor. Use Localization(LocalizationAsset, string) instead.
  • Localization.ChangeLanguage(string). Use currentLanguageName or currentLanguageCode instead.
  • Localization.ChangeLanguage(SystemLanguage). Use currentLanguageName or currentLanguageCode instead.
  • LocalizationSettings.loadSystemLanguage. The end user's system language is always loaded as the initial language.

REMOVED
  • Editor: Removed the 'Language Options' for the selected language from the toolbar. (The options are located in the Language window for each language)
